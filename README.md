# Bitcube QA Assessment

## Overview
This repository contains the QA assessment submission for the Factorial Calculator application.

## Contents
- Manual test cases
- Defect documentation
- Automated UI and API validation tests using Playwright with Java

## Automated Test Scenarios
1. Form validation styling
2. Factorial calculation for 12 (expected result: 479001600)
3. API request verification for correct headers and parameters

## Tech Stack
- Java
- Playwright
- JUnit 5
- Maven

## Project Structure

- `src/test/java/tests` - Test classes
- `docs` - Test cases and defects documentation

## How to Run
1. Clone the repository
2. Open in IntelliJ IDEA
3. Install Maven dependencies
4. Run tests using:
   ```bash
   mvn test
